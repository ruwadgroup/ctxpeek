## [1.0.0] — 2026-06-05

First stable release. The tool schema is now under semver. Highlights — full notes in the [package changelog](packages/ctxpeek/CHANGELOG.md):

### Added

- **`peek` grep mode** — pass a `query` to search inside a single named file (literal or `regex`) and get every match back with surrounding `context`. Deterministic, single-file navigation; no vector store.
- **`get_issues` filtering + read-one** — filter by `labels`, `author`, `assignee`, `since`, and `sort`/`order`; or pass a `number` to read one issue/PR's full body plus its top comments.

### Changed

- `peek` per-line (`max_line_length`) and match (`max_matches`) caps; binary and empty files are reported instead of dumped; no-match queries return a note rather than an error.
